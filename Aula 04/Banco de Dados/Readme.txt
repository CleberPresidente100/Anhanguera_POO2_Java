
* TABELA SQL

    A Query para se criar a tabela utilizada neste projeto está na pasta Scripts_BancoDados.


* CONECTOR MYSQL

    Para que o IntelliJ possa se conectar ao Banco de Dados, é necessário adicionar o Conector na Biblioteca do Projeto.
    Portanto é necessário executar oa seguintes passos:

        File --> Project Structure --> Project Settings --> Library --> New Project Library (+)

    Agora basta selecionar o conector que está na pasta "Conector JDBC MySQL" e pronto ! =D
